Introdução¶

QGIS é um popular software GIS open-source SIG com capacidades avançadas. Aqui você ira encontrar uma série de tutoriais e dicas Here is a series que mostram como utilizar o QGIS e como resolver alguns problemas comuns em SIG.

Você pode usar esses tutoriais como um curso básico sobre o software. No entanto, cada seção é independente, aqueles já familiarizados com QGIS podem saltar para qualquer seção.

Se você é professor, veja: doc:teach_qgis lá você encontra materiais e exemplos de como usar esses tutoriais em sala de aula.

Você também pode conferir: doc: learning_resources para outros materiais para ajudá-lo a aprender QGIS.
