Introduction¶

QGIS est un SIG populaire avec des fonctionnalités avancées. Voici une série de tutoriels et d’astuces qui vous montrent comment l’utiliser pour résoudre des problèmes communs liés aux SIG.

Vous pouvez utiliser ces tutoriels comme un cours d’auto-formation pour apprendre le logiciel complètement. Chaque section étant indépendante, ceux qui sont familiers avec QGIS peuvent cependant aller directement dans n’importe quelle section.

Si vous êtes éducateur, voir Teaching QGIS pour des ressources et des exemples d’utilisation de ces tutoriels afin d’enseigner des programmes liés aux SIG.

Vous pouvez également consulter QGIS Learning Resources pour d’autres ressources destinées à vous aider à apprendre QGIS.
