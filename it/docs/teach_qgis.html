These tutorials are available free of cost and under a very permissible license that allows educators to use these materials in classroom. Here are some examples of instructors using these tutorials to teach GIS.

  • Prof. John Barr is using these tutorials as assignments for Intro to GIS class at Ithaca College, New York.
  • Prof. Thomas Kearns uses some of the tutorials in his Urban Information Modeling class at IIT College of Architecture, Chicago.
  • Dr. Biju Soman is using these tutorials to teach the module on Public Health Technologies to the Master of Public Health Students at SCTIMST, Kerala, India.
  • Prof. Lex Berman is using some of the tutorials as part of QGIS Workshop at CGA, Harvard University.
  • Assist. Prof. Dr. Cengiz Yildirim is using the tutorials to teach Earth Information System at Istanbul Technical University.
  • Dr. Damien O’Grady of School of Earth and Environmental Sciences, James Cook University, Cairns, Australia is using the tutorials in his Introduction to GIS course.
  • R.S. Bhalla uses these tutorials as references for students in the An introduction to GIS and remote sensing using Quantum GIS at Foundation for Ecological Research, Advocacy and Learning (FERAL).
