속성작업¶

[ Download PDF A4 Letter ]

GIS 데이터는 두가지가 있습니다. 피처와 속성. 속성은 각 피처에 대한 구조데이터입니다. 이 튜토리얼에서는 QGIS에서 속성을 어떻게 보며 속성에 대한 기본 조회방법을 보여줍니다.

작업 개요¶

이 튜토리얼의 데이터셋은 세계의 인구분포에 대한 정보가 포함되어 있습니다. 작업은 전 세계를 대상으로 인구 1,000,000명을 초과하는 수도에 대한 조회와 찾기입니다.

데이터 획득¶

사용할 데이터는 Natural Earth의 Populated Places dataset 입니다.

`Natural Earth Populated Places shapefile. <http://www.naturalearthdata.com/http//www.naturalearthdata.com/download/10m/cultural/ne_10m_populated_places_simple.zip>`_를 다운받으십시오.

데이터 소스 [NATURALEARTH]

과정¶

  1. 다운로드가 완료되면 QGIS를 시작하십시오. 메뉴 선택에서 :menuselection:’레이어 –> 레이어추가 –> 벡터 레이어 추가...’

  1. Click on Browse and navigate to the folder where you downloaded the data.
  1. Locate the downloaded zip file ne_10m_populated_places_simple.zip. You do not need to unzip the file. QGIS has the ability to read zip files directly. Select the file and click Open.

  1. 오픈할 레이어를 선택하라는 창을 만나게 됩니다. 그러면 `ne_10m_populated_places_simple.shp’를 선택하고 :guilabel:’OK`를 클릭합니다.

  1. 선택된 레이어가 QGIS에 올라오고 세계지도에 인구분포를 나타내는 많은 점들을 보게 됩니다.

  1. 속성을 보기 위해 레이어에서 마우스 오른쪽 버튼을 클릭하고 :guilabel:`Open Attribute Table`를 선택합니다.

  1. 다양한 속성과 해당 값들을 살펴봅니다.

  1. 각 피처에서 인구에 관심이 있는데, `pop_max`가 찾는 필드입니다. 필드 헤더를 더블클릭하면 내림차순으로 정렬할 수 있습니다.

  1. 이제 이러한 속성에 대해 :guilabel:`Select features using an expression`으로 조회를 할 준비가 되었습니다.

  1. Select By Expression 창에서 Fields and Values`를 확장하고 `pop_max`를 더블클릭합니다. 그러면 아래쪽 표현식에 `pop_max`가 추가된 것을 알 수 있습니다. 만약 필드값이 확실하지 않다면 :guilabel:`Load all unique values 을 클릭해서 데이터셋에 나타난 값을 볼 수 있습니다. 이 예제에서는 1,000,000보다 큰 인구수를 갖는 피처를 찾고자 하는 것입니다. 그래서 표현식을 `“pop_max” > 1000000`와 같이 완성하고 :guilabel:’Select’을 클릭합니다.

  1. :guilabel:`Close`를 클릭하고 QGIS 메인으로 되돌아갑니다. 그러면 점 데이터의 일부가 선택되어진 것을 알 수 있습니다. 이것이 조회의 결과이고 데이터셋에서 인구가 1,000,000보다 큰 속성값을 가진 `pop_max`을 볼 수 있습니다.

  1. 이 예제의 목표는 각 나라의 수도를 찾는 것입니다. 이 데이터셋에서 각 나라의 수도만 다시 조회해 보도록 하겠습니다. 속성테이블에서 Select feature using an expression 버튼을 클릭합니다.

  1. 데이터는 `adm0cap`필드를 포함하고 있습니다. 값 ‘1’은 장소가 수도임을 의미합니다. 표현식을 `“adm0cap” = 1`라고 입력합니다. 바로 직전의 조회 결과 안에서만 데이터를 찾고자 하는 것이므로 :guilabel:`Select within selection’를 클릭합니다.

  1. :guilabel:`Close`를 클릭하고 QGIS 메인으로 되돌아갑니다. 그러면 보다 작 데이터셋이 선택된 것을 볼 수 있습니다. 이것이 두번째 조회 결과이며 데이터셋에서 인구가 1,000,000명을 초과하는 그 나라의 수도를 보여줍니다.

  1. 이 결과를 별도의 레이어로 저장해 봅시다. 레이어에서 마우스 오른쪽 버튼을 클릭하고 :guilabel:`Save Selection As`를 선택하십시오.

  1. `ESRI 파일 형식 ESRI Shapefile’을 유지하고 출력파일을 `large_capital_cities.shp`로 새 이름으로 저장 합니다. :guilabel:`Add saved file to map’박스에 클릭하고 :guilabel:’OK’를 클릭합니다.

  1. 새롭게 만들어진 세이프파일이 자동적으로 QGIS로 로딩됩니다. ne_10m_populated_places_simple 레이어를 해제합니다. 그러면 인구 1,000,000명을 초과하는 세계의 수도가 포함된 새로운 레이어를 보게 될 것입니다.
