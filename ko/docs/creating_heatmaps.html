열지도 만들기¶

[ Download PDF A4 Letter ]

열지도는 밀집한 점데이터를 가장 훌륭하게 시각화하는 도구중의 하나입니다. 열지도는 활동이 가장 활발하게 집중되어 집단을 만든 지역을 구분하여 쉽게 찾는데 사용되곤 합니다. cluster analysis 혹은 `hotspot analysis`를 수행하는데도 유용합니다.

과업 개요¶

이 예제에서는 2011년 영국 Surrey의 범죄발생위치 데이터셋으로 작업을 할 것이고 군에서 가장 범죄가 많이 발생한 지역을 찾을 것입니다.

데이터 획득¶

런던 데이터스토어는 `raw data from the Police.uk crime mapping website <http://data.london.gov.uk/datastore/package/policeuk-crime-data>`_를 제공합니다.

Download the Surrey data.

For convenience, you can directly download the sample data used in this tutorial from link below.

police-uk-crime-data-surrey.zip

과정¶

  1. 폴더에서 데이터 압축을 해제하고 시작합니다. 데이터는 CSV포맷으로 되어 있습니다. QGIS로 이 데이터를 불러옵니다(자세한 내용은 :doc:`importing_spreadsheets_csv`를 보십시오). 메뉴 레이어 –> 구분자로 분리된 텍스트 레이어 :menuselection:`Layer –> Add Delimited Text Layer`를 클릭합니다.

  1. 컴퓨터에서 police-uk-crime-data-surrey.txt`파일을 찾아서 엽니다. 파일 유형으로 :guilabel:`CSV (comma separated values)`를 선택합니다. *Easting*과 *Northing* 열이 자동적으로 X와 Y필드에 선택된 것을 볼 수 있습니다. 공간인덱스사용 :guilabel:`Use spatial index 옵션이 체크되어 있는지 확인합니다. 이것은 이 레이어의 작동속도를 높여줄 것입니다. :guilabel:`OK`를 클릭합니다.

  1. 몇개의 에러가 생깁니다. 이 예제에서는 에러를 무시할 수 있습니다. 닫기 :guilabel:`Close`를 클릭합니다.

  1. 다음, 좌표계 `Coordinate Reference System (CRS)`를 선택합니다. 데이터에 대한 자세한 설명을 읽으면 이 데이터의 공간참조가 British National Grid 임을 알게될 것입니다. CRS로 :guilabel:`OSGB 1936 / British National Grid`를 선택합니다. :guilabel:`OK`를 클릭합니다.

  1. 이제 QGIS로 데이터가 불러들여지는 것을 볼 것입니다.

  1. 데이터를 좀 더 자세히 보기 위해서 확대를 합니다. 데이터가 상당히 조밀하고 점들이 어디에 가장 밀집되어 있는지 알기 어렵다는 것을 알게될 것입니다. 이것이 어디에 열지도가 유용한지를 알려줍니다.

  1. To create the heatmap, you need to enable a core plugin named Heatmap. See 플러그인 사용하기 to know how to enable built-in plugins. Once you have enabled the plugin, go to Raster ‣ Heatmap ‣ Heatmap.

경고

The Heatmap plugin has a bug in QGIS 2.8.1. See this post for workarounds.

  1. 열지도 플러그인 :guilabel:`Heatmap Plugin`에서 출력 래스터 :guilabel:`Output raster`sms 파일이름으로 `crime_heatmap`를 입력합니다. 반지름 :guilabel:`Radius`으로 `1000`을 입력합니다. 반지름은 각 포인트의 주변지역으로 픽셀이 갖고 있는 `heat`를 계산하는게 사용될 것입니다. 고급 :guilabel:`Advanced`을 체크해서 열지도의 출력 크기를 정할 수 있습니다. 셀크기 X :guilabel:`Cell Size X`와 셀크기 Y :guilabel:`Cell Size Y`로 `100`을 입력합니다. :guilabel:`OK`를 클릭합니다.

  1. 일단 과정이 종료되면 캔버스에 회색 열지도를 보게 될 것입니다.

  1. 열지도를 자주 보아왔던 전통적인 열지도로 만들어 보겠습니다. 열지도 레이어를 우측클릭하고 속성 :guilabel:`Properties`을 클릭합니다.

  1. 스타일 Style`탭에서 랜터유형 :guilabel:`Render type`으로 단일밴드 의사색채 :guilabel:`Singleband pseudocolor`를 선택합니다. 다음으로, 최소/최대값 불러오기 :guilabel:`Load min/max values 부분에서 정확도 Accuracy`로 실제값 느림 :guilabel:`Actual (slower)`를 선택하고 불러오기 :guilabel:`Load`를 클릭합니다. 이 작업은 열지도를 조사하고 최소 그리고 최대 픽셀값을 찾습니다. 이러한 값들은 적절한 색상표를 만드는데 사용될 것입니다. 새 색상표 작성 :guilabel:`Generate new color map`부분에서 색상표로 :guilabel:`YlOrRd (Yellow-Orange-Red)를 선택하고 분류 :guilabel:`Classify`를 클릭합니다. :guilabel:`OK`를 클릭합니다.

  1. 이제 보다 열지도 답게 표현된 레이어를 보게될 것입니다. 객체 확인 :guilabel:`Identify`툴을 선택해서 열지도의 아무 픽셀이나 클릭을 합니다. 결과 팝업창에서 픽셀 값을 보게될 것입니다. 이 픽셀 값은 픽셀 주변 반경 즉, 명시된 반경 내(예제에서는 - 1000m)에 원래 레이어로부터 얼마나 많은 점들이 포함되어 있는지를 계산한 것입니다.

  1. 이제 열지도가 완성되었습니다. 이것은 시각적 해석에 유용합니다. 그러나 분석에 이 결과를 사용하기에는 유용하지 않습니다. 여러번 가장 점이 밀집한 곳 즉 hotspots 을 찾아내고자 했습니다. 이 열지도를 가지고 hotspots 을 찾아보도록 하겠습니다. 메뉴 래스터 –> 래스터 계산기 :menuselection:`Raster –> Raster Calculator`로 가십시오.

  1. 첫번째로 임계값을 정해야 합니다. 임계값을 넘는 모든 픽셀값들은 집단으로 간주될 것입니다. 이 데이터에서는 5를 사용해 보겠습니다. 래스터 계산기 Raster calculator`에서 출력 레이어로 `crime_hotspots`로 파일이름을 입력합니다. 래스터밴드 :guilabel:`Raster bands`부분에서 :guilabel:`crime_heatmap@1 를 더블클릭하면 래스터 연산 표현식 Raster calculator expression 텍스트 부분에 추가될 것입니다. “crime_heatmap@1” > 5`로 표현식을 완성합니다. 결과를 프로젝트에 추가 :guilabel:`Add result to project 옆 상자를 체크하고 :guilabel:`OK`를 클릭합니다.

  1. QGIS에 새로운 레이어가 추가될 것입니다. 이 레이어의 픽셀들은 0 혹은 1의 값을 갖습니다. 픽셀값이 5보다 큰 입력레이어의 모든 픽셀들은 1의 값을 갖고 나머지 픽셀들은 0이 됩니다. 메뉴 래스터 –> 변환 –> 폴리곤화 /래스터를 벡터로 :menuselection:`Raster –> Conversion –> Polygonize (Raster to Vector)`로 갑니다.

  1. 출력 파일명을 crime_hotspots_vector`로 입력합니다. 필드이름 :guilabel:`Field name`뿐만 아니라 종료시 캔버스로 불러옴 :guilabel:`Load into canvas when finished 옆 상자를 체크합니다. :guilabel:`OK`를 클릭합니다.

  1. 일단 변환이 종료되면 또 다른 레이어가 QGIS에 추가될 것입니다. 이것은 전단계에서 만들어진 클러스터가 벡터로 표현된 것입니다. 이 레이어는 0과 1값 모두를 가진 클러스터를 포함하고 있습니다. 0값을 걸러내 보도록 하겠습니다. 그러면 핫스팟 클러스터를 얻게됩니다. 레이어를 우측클릭하고 속성테이블 열기 :guilabel:`Open Attribute Table`를 선택합니다.

  1. 속성 테이블 :guilabel:`Attribute table`에서 표현식을 이용해 객체 선택 :guilabel:`Select feature using an expression`을 클릭합니다.

  1. 표현식으로 `“DN” = 1`를 입력하고 선택 :guilabel:`Select`을 클릭합니다. 다음 닫기 :guilabel:`Close`를 클릭합니다.

  1. QGIS 메인 창에서 몇 객체가 노란색으로 빛나는 것을 보게 될 것입니다. 이것들이 조회와 맞게 선정된 객체들입니다. 레이어를 우측클릭하고 다른 이름으로 저장 :guilabel:`Save Selection As...`를 선택합니다.

  1. 출력 레이어로 파일이름을 crime_clusters`로 입력합니다. 저장된 파일을 지도에 추가 :guilabel:`Add saved file to map 옆 상자를 체크하고 :guilabel:`OK`를 클릭합니다.

  1. 완료되었습니다. 최종 레이어는 열지도에서 추출한 `hotspots`을 포함하고 있습니다. 이 클러스터는 원자료로부터 모아진 *intelligence*이고 유용한 시각을 제공할 뿐만 아니라 추가 행동을 위한 기초입력자료로 제공될 수 있습니다.
